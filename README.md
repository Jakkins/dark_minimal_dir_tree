## installation

```bash
pip install mkdocs-dark-minimal-dirtree
python -m mkdocs new proj1
code proj1/
nano mkdocs.yml
```

```yml
site_name: My Docs
theme:
  name: dark_minimal_dirtree
```

```bash
python -m mkdocs serve
```
